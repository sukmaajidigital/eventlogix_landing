Arsitektur Sistem

Dibangun untuk skala, didesain untuk setiap acara.

Empat modul inti yang berjalan di balik setiap undangan — dari infrastruktur domain hingga check-in di lokasi acara.

Module / Domain

Multi-Tenant & Domain Kustom

Setiap event otomatis mendapat subdomain sendiri (namaevent.eventlogix.id) begitu dibuat. Ingin pakai domain milik klien? Arahkan CNAME atau A Record, dan SSL terpasang otomatis — tanpa perlu tim IT.

CNAME · A Record · Auto-SSL

Module / Theme

Dynamic Theming Engine

Susun undangan dari blok modular — Hero, Galeri, Kisah, RSVP — lalu simpan strukturnya sebagai JSON. Satu mesin tampilan, kombinasi desain yang tidak terbatas untuk setiap jenis acara.

Block-based · JSON Schema

Module / RSVP

RSVP & Manajemen Tamu yang Fleksibel

Form RSVP tidak kaku. Tambah atau kurangi field — teks, pilihan, radio — sesuai kebutuhan acara: jumlah pax untuk pernikahan, dietary restriction untuk seminar, konfirmasi after-party untuk ulang tahun.

EAV-based · Text · Select · Radio

Module / Check-in

Tautan & QR Code per Tamu

Setiap tamu menerima tautan personal (?to=Nama+Tamu) lengkap dengan QR Code uniknya sendiri — siap dipakai untuk proses check-in yang cepat dan akurat di lokasi acara.

Personalized Link · QR Check-in
